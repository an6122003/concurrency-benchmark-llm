# Techies Lab Benchmark Brand Asset Pack

This folder contains decorative assets for the AI hardware benchmark website and reports.

## Pixel Pet Benchmark Poses

- `pixel-pet/pet-gpu-trophy.png`: use for hardware result hero, GPU profile pages, winner cards.
- `pixel-pet/pet-analytics-dashboard.png`: use for charts, suite reports, comparison dashboards.
- `pixel-pet/pet-server-rack.png`: use for local server, runtime/backend, deployment pages.
- `pixel-pet/pet-token-courier.png`: use for throughput, tokens/sec, concurrency pages.
- `pixel-pet/pet-solution-architect.png`: use for solution architecture, checklist, buyer guide pages.

## Brand Copy

Short labels:

- Small. Curious. Limitless.
- Benchmark the local AI stack.
- Find the hardware your team can actually use.
- Tokens, latency, context, confidence.
- From GPU specs to team-ready answers.

Section titles:

- Local AI, measured for real teams.
- The curious way to compare hardware.
- Not just peak tok/s.
- Serve the team, not the spec sheet.
- Where models, runtimes, and GPUs meet.

Empty states:

- Pixel Pet is waiting for benchmark data.
- Add a result JSON to light up the dashboard.
- No hardware matched this filter. Try widening the search.

CTA copy:

- Run a benchmark
- Compare results
- Import result JSON
- Open suite report
- Explore hardware

Usage guidance:

- Keep assets on dark or violet-glow backgrounds.
- Use Pixel Pet as a helper/guide, not as a distracting hero on every section.
- Prefer small avatar/card placements for dense database pages.
- Use the GPU trophy pose sparingly for winners or highlighted hardware.
- Use analytics/dashboard pose near charts and reports.
- Use solution architect pose near buyer guidance or deployment recommendations.
